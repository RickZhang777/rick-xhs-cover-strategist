# Cover Methodology

Use this reference when choosing cover hooks, cover text, title structure, or when diagnosing why a cover is readable but not clickable.

## Core Formula

Click potential = relevance x hook x readability.

- Relevance: the target user should recognize "this is about me" within half a second.
- Hook: the cover needs contrast, number, suspense, emotion, loss, contradiction, or a strong promise.
- Readability: the main text must be readable in a mobile feed thumbnail.

If readability is already acceptable, prioritize relevance and hook.

## Cover Text vs Post Title

Do not treat the cover text as a chopped-up version of the post title.

- Cover text: emotional trigger, very short, creates "why?" or "is that true?"
- Post title: search/context entry, longer, explains the topic direction.

Good relationship:

- Cover text = contradiction or tension.
- Post title = who this is for and what the content explains.

Example pattern:

```text
Cover: 不忙 不等于 舒服
Title: 我发现舒服的工作都有几个共同点
```

## Strong Hook Types

Prefer these hook types when the topic supports them:

1. Contradiction / paradox
   - `A 不等于 B`
   - `越 A 越 B`
   - `做了 X 却 Y`
   - `你以为 A，其实 B`

2. Loss / cost
   - Words like `白干`, `踩坑`, `烧钱`, `亏了`, `智商税`, `越管越乱`.

3. Unique or scarcity judgment
   - `只有一个关键`
   - `就这一件`
   - `真正的问题是...`

4. Specific number
   - Concrete numbers are stronger than vague adjectives.
   - Use numbers only when the content can support them.

5. Suspense or forbidden knowledge
   - `真相`, `潜规则`, `没人说`, `你中招了吗`.

Avoid generic cover text:

- `三个提升销售的方法`
- `创业必须避开的坑`
- `产品为什么卖不动`

These are understandable, but often lack tension.

## Audience Anchor

Put the audience anchor as early as possible in the cover or title.

For business/consulting/founder content, useful anchor words include:

- 老板
- 生意
- 获客
- 成交
- 合伙人
- 外包
- 客服
- 团队

If the target audience is not business owners, adapt the anchor to the actual group.

## Visual Implications

- Real person photos usually help trust when personal IP matters.
- Realistic scenes such as bookshelf, whiteboard, office, teaching, or work context can strengthen credibility.
- Whiteboard and framework scenes are useful for dense business or methodology content.
- Pure text cards can work only when the text itself is very strong.
- Do not overload the cover with small labels; the main hook should dominate.

## Output Guidance

When generating title options, separate:

- Cover text: 3-8 Chinese characters or 2 short lines when possible.
- Post title: around 15-24 Chinese characters, can include audience keyword, number, or search term.

When the user asks for a cover prompt only, still include the cover text strategy inside the visual prompt.
